# Client / Recruiter Pitch — 5 phút

Script ngắn khi present portfolio (interview, khách hàng, Upwork call).

---

## 1. Mở đầu (30s)

> "Tôi xây portfolio AWS hands-on gồm 3 hướng: kiến trúc cloud classic theo chuẩn SAA-C03, GenAI production trên Bedrock, và AI agent tự động hóa UI. Mỗi project có architecture doc, IaC, và demo khi hoàn thành."

---

## 2. Chọn 1–2 project theo đối tượng (3 phút)

| Người nghe | Demo project | Nói gì |
|------------|--------------|--------|
| Cloud Architect hiring | 3-Tier HA + DR | Multi-AZ, failover, RPO/RTO |
| AI/ML team | Financial Audit Agent | Tool use, không để LLM tự tính số |
| Startup founder | CRM Automation | Chat tiếng Việt → tự làm việc trên web |
| Enterprise IT | Enterprise RAG | KMS, WAF, Guardrails, data không train model |

**Cấu trúc mỗi project (1.5 phút):**
1. Problem — khách/team gặp gì
2. Architecture — diagram, 3–4 services chính
3. Trade-off — tại sao chọn X không chọn Y
4. Demo — live hoặc screenshot

---

## 3. Đóng (30s)

> "Repo có 15 project được document — có thể đi sâu bất kỳ phần nào. Phần [X] gần nhất với nhu cầu team mình."

---

## Checklist trước khi present

- [ ] GitHub repo public hoặc share link read-only
- [ ] README root + PROJECT-INDEX cập nhật status
- [ ] Demo chạy được hoặc video backup
- [ ] Không show credentials / account ID thật trên slide
- [ ] Chuẩn bị teardown nếu demo live trên AWS

---

## Tài liệu kèm theo (gửi sau meeting)

1. Link GitHub repo
2. `portfolio/PROJECT-INDEX.md`
3. Architecture diagram project liên quan
4. (Optional) CV 1 trang
