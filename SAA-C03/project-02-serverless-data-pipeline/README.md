# Project 2: Serverless Data Processing Pipeline

Upload file → Lambda parse → DynamoDB → API Gateway + Cognito query.

**Trạng thái:** Chờ triển khai

## Mục tiêu

- S3 upload trigger Lambda
- Parse CSV/JSON → ghi DynamoDB
- REST API (Cognito auth) đọc dữ liệu

## AWS Services

S3, Lambda, DynamoDB, API Gateway, Cognito

## SAA-C03 focus

Serverless, cost-optimization, high-performance
