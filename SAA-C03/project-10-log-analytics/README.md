# Project 10: Centralized Log Analytics

- EC2 logs → CloudWatch Logs
- Subscription filter → Kinesis Firehose → OpenSearch

EC2, CloudWatch Logs, Kinesis Data Firehose, OpenSearch Service

## SAA-C03 focus

Real-time streaming, analytics, operational troubleshooting
